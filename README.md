# MNCA - simple visualization in CppFront (C++2 Syntax)

## Description
Simple proof-of-concept of a visualization of multiple neighborhoods cellular automata (MNCA) in Cpp2. 
It uses [Raylib](https://raylib.com/) as a drawing library.

## Dependencies
- C++20 compiler
- Make
- git

## Build
```bash
make
```

## Run
```bash
./mnca
```
